
# Take-Home Assignment: Complete Website with Advanced HTML & CSS

## Assignment Title:

**“Personal Portfolio Page – Full HTML & CSS Project”**

---

# Objective

Create a fully structured webpage using:

* Semantic HTML
* Forms
* Tables
* Embedded video
* Complete CSS styling
* Proper cascading
* Clean structure and correct syntax

This project tests both HTML knowledge (from W3Schools) and CSS mastery.

---

# Project Structure

```
portfolio-project/
│
├── index.html
└── styles.css
```

---

# Part 1: HTML Requirements (index.html)

Your webpage must include the following:

## 1. Semantic Structure (Required)

Use proper semantic tags:

* `<header>`
* `<nav>`
* `<main>`
* `<section>`
* `<article>`
* `<aside>`
* `<footer>`

Your layout should be logically structured.

---

## 2. Headings and Text

* One `h1`
* At least two `h2`
* At least three paragraphs

---

## 3. Navigation

Inside `<nav>`:

* At least 3 links (`<a>`)
* Links can point to `#section` anchors

---

## 4. Table

Create a table that includes:

* `<table>`
* `<tr>`
* `<th>`
* `<td>`
* At least 3 rows of data
* A table caption (`<caption>`)

Example idea: Weekly schedule, skills list, or grades table.

---

## 5. Form

Create a contact form that includes:

* `<form>`
* Text input
* Email input
* Password input
* Radio buttons
* Checkbox
* Dropdown (`<select>`)
* Textarea
* Submit button

Use proper `label` elements for accessibility.

---

## 6. Video

Embed one video using:

* `<video>` with controls
  OR
* An embedded YouTube iframe

---

## 7. Required Attributes

* One element with an **id**
* Two elements sharing the same **class**
* One element with **two classes**
* Proper `alt` attribute if using images

---

## 8. Comments

Include:

* At least 2 HTML comments

---

# Part 2: CSS Requirements (styles.css)

All major styling must be in the external stylesheet.

You must include:

## 1. Universal Selector

Add margin or padding reset.

## 2. Element Selectors

Style:

* `body`
* `p`
* `table`

## 3. Class Selector

Style multiple elements using one class.

## 4. ID Selector

Style one unique section differently.

## 5. Grouping Selector

Example:

```
h1, h2, h3
```

## 6. Descendant Selector

Example:

```
section p
```

## 7. Attribute Selector

Example:

```
input[type="text"]
```

## 8. Hover Effect

Add hover styling to navigation links.

## 9. Table Styling

Add:

* Borders
* Padding
* Background color for header row

## 10. Form Styling

Style:

* Input fields
* Submit button
* Add spacing between elements

---

# Part 3: CSS Application Rules

You must demonstrate:

## External CSS

Main styling in `styles.css`.

## Internal CSS

Inside `<style>` in the head:

* Change background color of the page
  OR
* Change one heading style

## Inline CSS

Use inline styling on ONE element only.

---

# Part 4: Cascading Demonstration

You must:

* Style all paragraphs using `p { }`
* Override one paragraph using a class or id
* Show that more specific selectors take priority

---

# Part 5: Avoid These Errors

Do not:

* Forget semicolons
* Leave braces unclosed
* Misspell CSS properties
* Write values like `20 px`
* Forget to close HTML tags
* Mix up class and id

---
